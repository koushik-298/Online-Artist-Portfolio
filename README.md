Sienna Muse — High-End Fine Art Portfolio

A static online portfolio showcasing a luxury fine art photography and mixed-media design concept. The site is built with:

- `index.html` — handcrafted portfolio landing page and content structure
- `index.css` — custom styling and theme tokens
- `app.js` — interactive portfolio behaviors, navigation, and content data
- `assets/` — imagery and visual assets used across the portfolio

## Features

- Premium portfolio experience with ambient glassmorphism, organic textures, and elegant typography
- Responsive landing page with smooth desktop and mobile navigation
- Interactive gallery sections, exhibitions, journal posts, and contact/inquiry interactions
- Tailwind CDN for utility-first layout plus custom theme extensions
- Custom cursor, mobile menu, sticky header, and animated hero interactions

## Getting Started

### Option 1: Open locally

Open `index.html` in your browser to view the portfolio locally.


## Project Structure

- `index.html` — main portfolio page
- `index.css` — styling and custom theme variables
- `app.js` — portfolio interaction logic and content data
- `server.js` — optional demo server (not required for portfolio preview)
- `assets/` — images, icons, and visual resources
- `package.json` — project metadata and scripts

## Notes

- This repository is primarily a static portfolio site; no backend is required for the portfolio content.
- If you want to preview the portfolio in a browser without using npm, opening `index.html` directly works fine.
- The current `package.json` also includes a demo OTP server script, but the portfolio itself is static.
