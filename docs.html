06. Maintenance & Expansion Guide

The Sienna Muse application has been built utilizing vanilla modern technologies to guarantee incredibly fast page loading and clean file organization. Follow the guidelines below to update site items.

How to add a new Artwork piece

Open the app.js file and locate the ARTWORKS array constant. Create a new object entry containing unique properties: id, title, category (Botanical, Ethereal, Still Life, Commissions), medium, year, dimensions, price, status (Available, Sold, Commissioned), image (relative assets directory path), popularity (0-100 rating for sorting), description, and notes. The extended portfolio grid dynamically builds and links detailed specs on the fly!

Modifying style tokens and palette shifts

Core theme variables (primary rusts, olive shades, and fonts) are set inside the Tailwind CSS configuration block in the head of index.html. Modifying color hex codes inside the theme.extend.colors block will update styling variables globally across all inline class representations automatically.
